## 🛠️ Getting Started

Follow these steps to unleash the dragon:

<details>
<summary><b>1️⃣ Prerequisites (Click to expand)</b></summary>
<br>

* **VirtualBox**: Ensure you have [Oracle VM VirtualBox](https://www.virtualbox.org/wiki/Downloads) installed.
* **Disk Image**: Download the corresponding Kali Linux `.vdi` (Virtual Disk Image) for version 2026.1.
</details>

<details>
<summary><b>2️⃣ Installation (Click to expand)</b></summary>
<br>

1. Clone or download this repository.
2. Ensure your `.vdi` file is accessible on your host machine.
3. Open VirtualBox.
4. Go to `Machine` -> `Add...` (or press `Ctrl+A`).
5. Browse and select the `kali-linux-2026.1-virtualbox-amd64-1.16-1.16.vbox` file from this repository.
</details>

<details>
<summary><b>3️⃣ Configuration (Click to expand)</b></summary>
<br>

* Right-click the newly added VM in VirtualBox and select **Settings**.
* Navigate to **Storage** and ensure the SATA Controller is pointing to your downloaded `.vdi` file.
* Adjust **System** (RAM, CPU cores) settings according to your host machine's capabilities for optimal performance.
</details>

---

## 🚀 Launch

Hit the **Start** button in VirtualBox and step into the matrix. Happy Hacking! 💻🕵️‍♂️
