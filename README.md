# Kali VirtualBox Setup

This repository contains the VirtualBox configuration file for Kali Linux 2026.1.

## Contents
- `kali-linux-2026.1-virtualbox-amd64-1.16-1.16.vbox`: The VirtualBox machine configuration file.

## Usage
1. Make sure you have the corresponding `.vdi` (Virtual Disk Image) file for Kali Linux 2026.1 downloaded.
2. Open VirtualBox and add the `.vbox` file to your list of virtual machines by going to `Machine -> Add` and selecting the file.
3. If necessary, adjust the storage settings in VirtualBox to point to your `.vdi` file.
